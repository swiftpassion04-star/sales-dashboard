# Sales Dashboard

This repository contains the existing sales dashboard plus the Project CRM dashboard.

## Project CRM Streamlit Cloud

Use this main file path in Streamlit Cloud:

```text
crm_dashboard.py
```

Set these Streamlit Cloud secrets:

```toml
CRM_SUPABASE_URL = "https://hilncnvpmroslecwzsns.supabase.co"
CRM_SUPABASE_ANON_KEY = "YOUR_ANON_KEY"
```

Do not add the Supabase service role key to this repository or to Streamlit Cloud.
